Business Directory r19.0 README
==========================

Thank you for downloading Business Directory distribution.

This distribution includes the source code of the Business Directory app in 4 versions; Ionic 1.x, Ionic 3.x, Ionic 4.x and Ionic 5.x. It also includes Business Directory Firebase Admin, an  Angular web app. All of them are ready to run (almost) out the box.

Overview
--------

Specifically, your pack consists of the following three
directories:

    * [Business Directory 1]
        Documentation and source code of the Business Directory app based on Ionic 1.x.
    * [Business Directory 2]
        Documentation and source code of the Business Directory app based on Ionic 3.x (2nd Generation).
    * [Business Directory 4]
        Documentation and source code of the Business Directory app based on Ionic 4.x.
    * [Business Directory 5]
        Documentation and source code of the Business Directory app based on Ionic 5.x.
    * [Business Directory Firebase Admin ]
        Documentation and source code of the Admin panel web app.
